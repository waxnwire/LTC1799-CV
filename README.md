# LTC1799 CV
CV controlled LTC1799

This is a circuit schematic, PCB and Gerber files (for creating your own)

LTC1799 Clock - perfect for circuit bending

Unlike other LTC1799 boards out there, this one allows for two CV inputs

Set the base pitch, and then using an LFO or Sequencer, modulate the pitch

# Cost:
while boards from Etsy etc might cost around $10each, but learning how to manufacture the boards yourself, it should cost around the same each (making 5), but with the bonus value of it being a voltage controlled clock!
